# Ruboty::Rhymer

[rhymer](https://github.com/suzuki86/rhymer) handler for [Ruboty](https://github.com/r7kamura/ruboty).

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'ruboty-rhymer'
```

## Usage

```
$ bundle exec ruboty
Type `exit` or `quit` to end the session.

> 今日はとても良い天気ですね。
> いや、そうでもないか？
> でも、こんな日は自然に元気になります。
今日は良い天気　こんな日は自然に元気
```
